# Passport Strength Analysis

## Project Overview

This project analyzes global passport strength using visa-free access data to explore how mobility varies across countries and regions. The goal was to quantify travel freedom patterns and uncover geographic disparities using data analysis and visualization techniques.

---

## Problem Statement

How does passport strength vary globally, and what regional or structural patterns explain differences in international travel freedom?

---

## Data

- Source: Public passport strength / visa access datasets
- Scope: Global country-level data
- Key variables:
  - Number of visa-free destinations
  - Passport ranking
  - Country / region
  - Additional comparative indicators

---

## Methodology

### 1. Data Cleaning
- Standardized country names
- Handled missing or inconsistent values
- Verified numeric integrity of visa-access counts

### 2. Exploratory Data Analysis
- Analyzed distribution of passport strength globally
- Grouped results by region
- Compared top and bottom ranking countries

### 3. Visualization
- Built comparative bar charts and geographic visualizations
- Highlighted regional clusters and inequality in mobility access

---

## Key Findings

- Significant disparities exist in global mobility access
- Regional clustering patterns are clearly visible
- A small number of countries dominate the top passport rankings
- Lower-ranking passports show strong geographic concentration
